# Python print() 함수 정리

## 1. print() 함수의 매개변수 연결방법
`print()`함수는 소괄호 안에 여러개의 매개변수를 넣어 값을 출력 할 수 있다. 이때 값들을 어떻게 연결하느냐에 따라 출력 모양이 달라진다.

### 1-1. 콤마(`,`)로 연결하기
- **특징:** 콤마로 값들을 연결하면 출력할 때 값 사이에 자동으로 `space`가 생긴다.
- 서로 다른 자료형(문자열과 숫자: 원래는x)을 한 번에 연결해서 출력 할 수 있다.

```python
day = "목요일"
print("오늘은", day, "입니다.")

# 출력 결과: 오늘은 목요일 입니다.
```
### 1-2. 더하기(`+`)로 연결하기
- **특징:** 더하기로 연결하면 공백없이 완전히 붙어서 출력 된다.
- **주의사항:** 서로 다른 자료형(문자열과 숫자)을 더하기로 연결하면 파이썬은 `TypeError`를 발생시킨다.

 ```python
day = "목요일"
print("오늘은" + day + "입니다.")

# 출력 결과: 오늘은목요일입니다.
```
## 2. 사용자 입력값의 데이터 타입 문제와 해결 과정
`input()`함수로 입력받은 값은 화면에 숫자로 보이더라도 실제로는 문자열(str)타입이다. 이 문제를 확인하고 해결하는 단계를 정리한다.

### 2-1. type() 함수로 변수의 정체 확인하기
 - `type()`함수는 변수 안에 들어있는 데이터의 자료형을 알려주는 함수이다.

```python
number = input("숫자를 입력하세요: ")  # 사용자가 100을 입력했다고 가정
print(type(number))

#  출력 결과: <class 'str'>
```

### 2-2.[1단계] 문자열 상태에서 계산을 시도할 때의 오류
문자열 타입인 `number`변수에 숫자`10`을 더하려고 하면 에러가 발생한다.

```python
number = input("숫자를 입력하세요: ")  # 100 입력
print(number + 10)                   # 에러 발생!
```
- **에러 메세지 원인 분석:** `TypeError: can only concatenate str (not "int") to str`
  - 이 에러는 문자열은 문자열 끼리만 연결 될 수 있다는 뜻이다.

### 2-3.[2단계] 출력할 때 정수로 임시 변환하기 (int 함수 활용)
출력하는 순간에만 `int()`함수를 씌워 임시로 정수로 바꾸어 연산하는 방법이다.

```python
number = input("숫자를 입력하세요:")  # 100 입력
print(int(number) + 10)

# 출력 결과:
# 숫자를 입력하세요:
# 110
```
### 2-4.[3단계] 입력 단계부터 정수로 변환하여 저장하기(가장 권장)
`input()`함수 전체를 `int()`로 감싸서 변수에 저장할 때부터 정수형 변수로 만드는 방법이다. -> 코드 간결

```python
number = int(input("숫자: "))  # 100 입력
print(type(number))
print(number * 10)

# 출력 결과:
# 숫자:
# <class 'int'>
# 1000
```

## 3. 실전 응용 문제
### 문제 1: 사칙연산 계산기 만들기
사용자로부터 두 개의 숫자를 각각 입력받아, 두 수의 합(더하기)과 곱(곱하기)을 출력하는 코드를 작성한다.
- **조건:**
  1. 입력 단계에서 `int(input())`을 사용하여 정수로 변환한다.
  2. 출력할 때는 콤마(`,`)를 사용하여 자연스러운 공백이 생기도록 만든다.

```python
x = int(input("x에 들어갈 숫자를 입력 하세요: "))   # 5 입력
y = int(input("y에 들어갈 숫자를 입력 하세요: "))   # 10 입력
print("x+y=", x+y, "이고", "x*y=", x*y, "이다.")

# f-string을 사용하면 아래와 같이 더 간결하게 표현할 수 있다.
# print(f"x+y= {x+y} 이고 x*y= {x*y} 이다.")

# 실제 출력 결과:
# x에 들어갈 숫자를 입력 하세요: 5
# y에 들어갈 숫자를 입력 하세요: 10
# x+y= 15 이고 x*y= 50 이다.
```
### 문제 2: 자기소개 카드 만들기
사용자에게 이름과 태어난 연도(정수)를 입력받아 내년 나이를 계산해 출력하는 프로그램을 만든다.
- **실행 예시:**

```python
이름을 입력하세요: 홍길동
태어난 연도를 입력하세요: 2005
홍길동 님의 내년 나이는 22 살 입니다.
```
- **조건:**
  1. 올해 연도는 2026년으로 고정해서 계산한다. (나이 계산 공식: 올해 연도 - 태어난 연도 + 1)
  2. 출력할 때 print()의 콤마(,)를 활용하여 문자열과 계산 결과를 함께 출력한다.

```python
name = input("이름을 입력하세요: ")  # 유은혜 입력
year_born = int(input("태어난 연도를 입력하세요: ")) # 1990년 입력
print(name, "님의 내년 나이는", 2026 - year_born + 1, "살 입니다.")

# 실제 출력 결과:
# 이름을 입력하세요: 유은혜
# 태어난 연도를 입력하세요: 1990
# 유은혜 님의 내년 나이는 37 살 입니다.
```
### 문제 3: 문자열 연결의 에러 해결하기 (더하기 연산 응용)
아래 코드는 에러가 발생한다. 이 코드가 정상적으로 `오늘 온도는 25도 입니다.`라고 출력되도록 형변환 함수를 사용하여 코드를 수정한다.

- **에러가 발생하는 코드:**

```python
temp = 25
print("오늘 온도는 " + temp + "도 입니다.")  # TypeError 발생!
```

```python
print("오늘 온도는", str(temp) + "도 입니다.")

# [풀이 및 해석]
# 1. "오늘 온도는"과 "25도" 사이에 공백(띄어쓰기)이 들어가야 하므로, 콤마(,)를 사용해 공백을 자동으로 만들어 준다.
#    (만약 더하기(+)를 사용해 연결하려면 "오늘 온도는 "과 같이 따옴표 안에 수동으로 공백을 1칸 넣어주어야 한다.)
# 2. temp = 25는 숫자(int)이므로 str()을 사용해 문자열로 형 변환을 해준다. 
#    이렇게 해줘야 더하기(+) 연산자를 사용해 문자열인 "도 입니다."와 에러 없이 바로 이어서 출력할 수 있다.

# 실제 출력 결과:
# 오늘 온도는 25도 입니다.
```
