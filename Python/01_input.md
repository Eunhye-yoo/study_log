# Python input() 함수 정리
## input() 함수란?

- 사용자로부터 값을 입력받는 함수이다.
- `input()` 함수의 매개변수는 사용자에게 보여줄 안내 문구이다.
- `input()` 함수에는 하나의 매개변수만 사용할 수 있다.
- `input()` 함수로 입력받은 값은 기본적으로 문자열(str)로 처리된다.
- 숫자로 사용하려면 `int()` 또는 `float()` 함수를 사용해야 한다.

## 형 변환(Type Casting)

### 문자열 → 숫자

```python
number = input("숫자 = ")
print(number * 20)
```

위 코드를 실행하면 사용자가 입력한 값을 문자열로 처리한다.
예를 들어 사용자가 "숫자 ="에 `5`를 입력하면 연산 결과인 `100`이 아닌, 문자열 `"5"`가 20번 반복된 결과가 출력된다.

```python
55555555555555555555
```

따라서, 산술 연산이 목적이라면 다음과 같이 입력 단계에서 `int()`함수를 사용해 정수형으로 변환해야 숫자 계산이 가능하다.

```python
number = int(input("숫자 = "))
print(number * 20)    # 출력: 100
```

### 숫자 → 문자열
서로 다른 자료형(int와 str)을 직접 더하려고 하면 파이썬은 예외를 발생시킨다.
- 원하는 결과: `20번째 접속하셨습니다.`

```python
n = 20
print(n + "번째 접속하셨습니다.")  # TypeError 발생
```

위 코드는 `n`을 숫자(int)로 처리하기 때문에 문자열(str)과 더할 수 없어 `TypeError`가 발생한다.
따라서 다음과 같이 `str()` 함수를 사용해 문자열로 변환해야 한다.

```python
n = 20
print(str(n) + "번째 접속하셨습니다.")
```

## 예시1

### 1-1. 문자열 입력 출력

```python
age = input("나이를 입력하세요 : ")
print("당신은", age + "살", "입니다.")
```

사용자가 `35`를 입력하면:

출력 결과:

```python
당신은 35살 입니다.
```
- `input()`으로 입력받은 값은 문자열(str)이다.
- 따라서 `age + "살"` 처럼 문자열끼리 연결할 수 있다.
- 참고: `print()` 함수 내에서 인자를 쉼표(`,`)로 구분하면 출력 시 인자 사이에 공백(띄어쓰기)이 자동으로 한 칸씩 삽입된다.
  
### 1-2. 흔히 하는 실수

```python
age = input("나이를 입력하세요 : ")
print("당신은 내년에", age + 1 + "살", "입니다.")  # TypeError 발생
```

위 코드는 오류가 발생한다.

`input()`으로 입력받은 값은 문자열(str)이기 때문에 숫자 `1`과 더할 수 없다.

따라서 `int()` 함수를 사용해 문자열을 숫자로 변환해야 한다.

```python
age = int(input("나이를 입력하세요 : "))
print("당신은 내년에", str(age + 1) + "살", "입니다.")
```
`age + 1`은 숫자(int)이기 때문에 문자열인 `"살"`과 바로 더할 수 없다.
따라서 `str()` 함수를 사용해 문자열로 변환해야 한다.

사용자가 `20`을 입력했을 때의 실제 출력 결과:

```python
당신은 내년에 21살 입니다.
```

### 1-3. f-string
문자열 ↔ 숫자 간의 빈번한 형 변환 과정은 코드를 복잡하게 만든다.
파이썬의 f-string을 사용하면 훨씬 직관적이고 간단하게 표현할 수 있다.

```python
age = int(input("나이를 입력하세요 : "))
print(f"당신은 내년에 {age + 1}살 입니다.")
```
- `f`를 문자열 앞에 붙이면 f-string 기능을 사용할 수 있다.
- 중괄호`{}` 안에는 변수나 계산식을 직접 넣을 수 있다.
- `{age + 1}` 부분이 먼저 계산된 후, 출력 시 자동으로 문자열로 변환되어 결합한다.
- 즉, 복잡한 `str()` 변환 과정을 파이썬이 내부적으로 처리해주므로 코드가 간결해진다.

## 예시2
### 2-1 실수(float) 입력 받기

```python
height = float(input("키를 입력하세요(cm): "))
print(f"당신의 키는 {height}cm 입니다.")
```

사용자가 아래와 같이 입력하면:

```python
173.5
```

출력 결과:

```python
당신의 키는 173.5cm 입니다.
```
- 키처럼 소수점이 포함될 수 있는 숫자를 다룰 때는 `float()` 함수를 사용한다.

**[참고] 코드 내부의 `int(173.5)`와 실행 중 사용자가 입력한 `int(input())`의 결정적 차이**

- **`int(173.5)` -> 정상 작동 (결과: `173`)**

  - 프로그래머가 소스코드 내에 진짜 숫자(실수)인 `173.5`를 직접 적어두고 정수로 바꿀 때는, 파이썬이 소수점 아래 자리를 버리고 정수 부분만 정상적으로 남겨준다.

- **사용자가 `input()`으로 `173.5`를 입력했을 때 `int(input())` -> `ValueError` 발생!**

  - 프로그램 실행 중에 사용자가 터미널에 입력한 값은 무조건 문자열(텍스트) 상태가 되므로, 파이썬은 이를 숫자 형태가 아닌 `"173.5"`라는 `'글자'`로 받아들인다.

  - 글자를 정수로 바꿀 때는 문장 안에 오직 깨끗한 숫자 형태만 있어야 하는데, 점(`.`)이 포함되어 있으면 파이썬이 숫자가 아닌 기호(문자)로 판단해 변환하지 못하고 에러를 뿜는다.
