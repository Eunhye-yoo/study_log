# Python input() 함수 정리
## input() 함수란?

- 사용자로부터 값을 입력받는 함수이다.
- input 함수의 매개변수는 사용자에게 보여줄 안내 문구이다.
- 하나의 input 함수에는 하나의 매개변수만 사용할 수 있다.
- input 함수로 입력받은 값은 기본적으로 문자열(str)로 처리된다.
- 숫자로 사용하려면 int() 또는 float() 함수를 사용해야 한다.

## 형 변환(Type Casting)

### 문자열 → 숫자

```python
number = input("숫자 = ")
print(number * 20)
```

위와 같이 출력하면 사용자가 입력한 값을 문자열로 처리한다.

예를 들어 사용자가 "숫자=" 에 5를 입력하면:

```python
55555555555555555555
```

처럼 문자열이 20번 반복 출력된다.

따라서,

```python
number = int(input("숫자 = "))
print(number * 20)
```

처럼 int() 함수를 사용해야 숫자 계산이 가능하다.

---

### 숫자 → 문자열

원하는 결과:

```python
20번째 접속하셨습니다.
```

```python
n = 20
print(n + "번째 접속하셨습니다.")
```

위 코드는 n을 숫자(int)로 처리하기 때문에 문자열과 더할 수 없어 TypeError가 발생한다.

따라서,

```python
n = 20
print(str(n) + "번째 접속하셨습니다.")
```

처럼 str() 함수를 사용해 문자열로 변환해야 한다.

## 예시

### 1. 문자열 입력 출력

```python
age = input("나이를 입력하세요 : ")
print("당신은", age + "살", "입니다.")
```

사용자가 아래와 같이 입력하면:
```python
35
```
출력 결과:
```python
당신은 35살 입니다.
```
- input()으로 입력받은 값은 문자열(str)이다.
- 따라서 `age + "살"` 처럼 문자열끼리 연결할 수 있다.

### 1-2. 흔히 하는 실수

```python
age = input("나이를 입력하세요 : ")
print("당신은 내년에", age + 1 + "살", "입니다.")
```

위 코드는 오류가 발생한다.

`input()`으로 입력받은 값은 문자열(str)이기 때문에 숫자 `1`과 더할 수 없다.

따라서 `int()` 함수를 사용해 문자열을 숫자로 변환해야 한다.

```python
age = int(input("나이를 입력하세요 : "))
print("당신은 내년에", str(age + 1) + "살", "입니다.")
```
`age + 1`은 숫자(int)이기 때문에 문자열인 `"살"`과 바로 더할 수 없다.
따라서 `str()` 함수를 사용해 문자열로 변환해야 한다.

사용자가 20을 입력하면:

```python
21살
```
※ 문자열 -> 숫자 -> 문자열의 변환은 코드를 복잡하게 만드므로 f-string 함수로 깔끔하게 정리할 수 있다※ 
```python
age = int(input("나이를 입력하세요 : "))
print(f"당신은 내년에 {age + 1}살 입니다.")
```

- `f`를 문자열 앞에 붙이면 f-string 기능을 사용할 수 있다.
- `{}` 안에는 변수나 계산식을 넣을 수 있다.
- `{age + 1}` 부분이 먼저 계산된 후 문자열 안에 자동으로 들어간다.
- f-string은 숫자를 자동으로 문자열로 변환해주기 때문에 `str()` 함수를 사용하지 않아도 된다.
- 즉, `str(age + 1)` 과 같은 문자열 변환 과정을 Python이 자동으로 처리해준다.
