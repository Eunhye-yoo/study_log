# 변수(variable)란?

- 값을 저장할 수 있는 메모리 공간을 의미한다.
- 변수명을 사용해 메모리에 저장된 데이터에 쉽게 접근하고 재사용할 수 있다.

## 기본 규칙(Naming Rules)
파이썬에서 변수명을 지을 때는 다음과 같은 필수 규칙과 권장 사항을 지켜야 한다.

### 1. 필수 규칙(위반 시 문법 에러 발생)
- **대소문자 구분**: `apple`과 `Apple`은 서로 다른 변수로 취급된다.
- **공백 불가능**: 변수명에는 공백(space)을 포함할 수 없다.
- **숫자 시작 불가**: 첫 글자는 반드시 영문자나 언더스코어(`_`)로 시작해야 하며, 숫자로 시작할 수 없다.
- **특수문자 제한**: 언더스코어(`_`)를 제외한 특수문자(예: `-`,`@`, `$`, `!`)는 사용할 수 없다.
- **예약어 사용 금지**: 파이썬에서 이미 고유한 기능으로 예약해 둔 단어(예: `class`, `if`, `for`, `import` 등)는 변수명으로 사용할 수 없다.

### 2. 권장 규칙 
- **직관적이고 명확한 이름 사용**: `a`, `b`, `tmp`처럼 역할을 알기 힘든 이름은 피하고, 저장하는 데이터의 의미를 담은 명확한 이름을 사용한다. (예: `user_id`, `total_price`)
- **네이밍 컨벤션 적용**: 여러 단어가 조합되어 변수명이 길어질 때는 가독성을 위해 특정 표기법(snake_case 등)을 일관되게 준수한다.
  
## 변수 표기법 (Naming Convention)

### snake_case (뱀 표기법)
단어와 단어 사이를 언더스코어(`_`)로 연결하는 방식이다. 

```Python
last_name = "Yoo"
first_name = "Eunhye"
```

### camelCase (낙타 표기법)
첫 단어는 소문자로 시작하고, 이어지는 단어의 첫 글자를 대문자로 표기하여 구분하는 방식이다. 주로 자바스크립트나 자바 계열에서 많이 쓰인다.

```Python
lastName = "Yoo"
firstName = "Eunhye"
```

## 잘못된 예시

```Python
my-name = "Yoo"  # 에러: 하이픈(-)은 사용할 수 없는 특수문자이며, 파이썬이 뺄셈 연산자로 오인함
1name = "Yoo"    # 에러: 숫자로 변수명을 시작할 수 없음
class = "Yoo"    # 에러: 'class'는 파이썬 내부 예약어(클래스 선언용)이므로 변수명 사용 불가능


```
